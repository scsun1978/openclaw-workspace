## 🎉 首个生产版本

Stock-SimGame Live 是一个实时股票模拟交易系统，支持限价单、撮合引擎、WebSocket 实时通信等功能。

---

## ✨ 功能清单

### 核心功能
- ✅ 用户注册/登录 (JWT 认证)
- ✅ 余额管理 (初始 100,000)
- ✅ 股票交易 (市价单/限价单)
- ✅ 撮合引擎 (价格优先/时间优先)
- ✅ 持仓管理
- ✅ 订单历史
- ✅ 交易记录

### 技术特性
- ✅ WebSocket 实时通信
- ✅ HTTP 轮询降级
- ✅ 自动重连 (指数退避)
- ✅ 心跳检测 (30s)
- ✅ Redis 缓存
- ✅ PostgreSQL 持久化

## 🐳 容器化部署

```yaml
services:
  app:      # Node.js 应用 (端口 3000)
  postgres: # PostgreSQL 15 (端口 5433)
  redis:    # Redis 7 (端口 6380)
```

### 部署命令

```bash
cd ~/stock-simgame-live
docker compose up -d --build
```

---

## ⚠️ 已知限制

1. **单机部署** - 无高可用架构
2. **内存订单簿** - 重启后未成交订单丢失
3. **限流建议** - 单机 ≤100 TPS
4. **Redis 依赖** - 需确保高可用
