# Stock-SimGame v2.0 前端 UI 设计规范

> **目标用户**: 青少年 (12-25岁)
> **设计风格**: 游戏化、惊艳、动态
> **核心原则**: 寓教于乐，让学习投资变得有趣

---

## 🎨 设计理念

### 核心关键词
- **🎮 游戏化** - 交易不再是枯燥的数字，而是一场冒险
- **✨ 惊艳** - 每个交互都有视觉反馈
- **🎉 奖励感** - 完成任务、达成成就时的满足感
- **🚀 动感** - 页面"活"起来，数据流动可见

### 色彩方案

### 2. 交互动效

| 交互 | 动效 | 说明 |
|------|------|------|
| **按钮悬停** | 脉冲发光 | 霓虹边框呼吸效果 |
| **按钮点击** | 波纹扩散 | 从点击位置扩散 |
| **卡片悬停** | 3D 倾斜 | 跟随鼠标轻微倾斜 |
| **页面切换** | 滑动 + 淡入 | 流畅过渡 |
| **数据更新** | 数字滚动 | 旧值 → 新值滚动动画 |
| **价格变动** | 颜色闪烁 | 红绿闪烁 0.5s |
| **加载中** | 脉冲骨架屏 | 呼吸效果占位 |

## 📱 响应式设计

### 桌面端 (≥1024px)
- 3 列布局
- 完整动效
- 大尺寸图表

### 平板端 (768px - 1023px)
- 2 列布局
- 保留核心动效
- 中等尺寸图表

### 手机端 (<768px)
- 单列布局
- 简化动效（性能优化）
- 手势操作优化

- **Howler.js** - 音频管理

### 图表
- **ECharts** - K 线图表
- **React-Confetti** - 庆祝效果

## ✅ 验收标准

1. **视觉冲击力**: 首次打开页面，用户有"哇"的感觉
2. **动效流畅度**: 60fps，无卡顿
3. **奖励感**: 完成任务/成就时，有明显满足感
4. **青少年友好**: 配色鲜艳、图标可爱、交互有趣
5. **性能**: 首屏加载 < 3s，LCP < 2.5s
