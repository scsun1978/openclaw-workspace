# Workload Dashboard

> **Dashboard UID**: `openclaw-workload`  
> **访问地址**: http://localhost:3000/d/openclaw-workload

## 用途

监控 OpenClaw 的**工作负载**，包括会话数、消息量、处理速度等。

## 核心指标

| 指标 | 含义 | 用途 |
|------|------|------|
| Active Sessions | 活跃会话数 | 监控并发量 |
| Message Rate | 消息处理速率 | 监控吞吐量 |
| Tool Calls | 工具调用次数 | 监控 API 使用 |
| Agent Runs | Agent 执行次数 | 监控 Agent 活动 |

## 使用场景

### 场景 1: 容量规划
- 分析峰值负载
- 规划资源扩容

### 场景 2: 性能监控
- 监控处理延迟
- 发现性能瓶颈
